# Harley

Kinect based application to help autistic children with mobility impairment and shape selection problems

## Features

 * Voice controlled interactions
 * Speech Synthesis for level engagements

## Activities

 * Star Jump
 * Shape Drawing
 * Facial Expressions
 * Karaoke
 * Hurdle Jump
